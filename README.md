# React Django Todo App

Simple CRUD Todo application built with:

- Django REST Framework (Backend)
- React (Frontend)
- Tailwind CSS (UI)

## Features
- Create todo
- Read todos
- Update todo
- Delete todo

## Project Structure

- backend/ → Django REST API
- frontend/ → React + Tailwind app
